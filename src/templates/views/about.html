¡Bienvenido a BTlogger!

BTlogger es el resultado de un apasionante proyecto llevado a cabo como parte del curso de Sistemas Inalámbricos en la prestigiosa Escuela Politécnica Nacional. Esta aplicación tiene un propósito claro y valioso:

Detectar dispositivos Bluetooth a nuestro alrededor y mostrar su disponibilidad en tiempo real.

¿Qué hace que BTlogger se destaque?

  • Capacidad para guardar dispositivos detectados en una base de datos.
  • Filtrado inteligente para mostrar solo dispositivos disponibles o fuera de rango.
  • Tecnologías avanzadas como Python, Flask, HTML, CSS y JavaScript.
  • Utiliza la potente librería PyBluez para realizar el escaneo Bluetooth.
  • Base de datos MySQL hospedada en Always Data para un almacenamiento seguro.
  • Integración perfecta con APIs como Bootstrap, FontAwesome, jQuery y Ajax.
